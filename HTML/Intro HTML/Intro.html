1. Wat is het verschil tussen het internet en het world wide web?
    Het internet is de verbinding tussen computers waar het wereld wijde web op werkt. Het wereld wijde web zijn de pagina's die je te zien krijgt.
  2. Wat is het verschil tussen HTML, CSS en JavaScript?
    HTML is om tekst in elementen onder te verdelen.
    CSS is om die elementen in een bepaalde stijl te stijlen.
    Javascript is om interactiviteit toe te voegen aan webpagina's.

    <head> is om metadata over de pagina te bevatten.

  7. Wat is het verschil tussen inline en block elementen?
    Een block element start atlijd op een nieuwe lijn en gebruikt de volle breedte van de pagina. Een inline element start niet op een nieuwe lijn en gebruikt alleen zoveel ruimte als nodig is.
