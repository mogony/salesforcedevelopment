1. Om een tabel te maken heb je volgende elementen nodig: Geef voor elke element een definitie.
    • table: hiermee definieer je een html table
    • groeperend elementen:
      • thead: dit groepeert de header van de tabel in een rij
      • tbody: dit geeft de basiscontent aan
      • tfoot: dit geeft de footercontent aan
    • th: tabelcel als een header
    • tr: tabelrij
    • td: tabelcellen
  2. Maak een tabel met <table> element.
    Voornaam Achternaam Leeftijd Telefoonnr
    Steven Smith 32 +31 12345678
    Alexander Beave 16 +31 12345678
