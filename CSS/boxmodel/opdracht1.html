Block en inline boxes gaan over hoe deze boxes zich gedragen ten opzichte van page flow. Een box can worden aangegeven met het display property en dan block of inline value.

Block boxes

  • Een block box begint op een nieuwe regel
  • Vult de gehele inline ruimte die over is in de container.
  • Width & height properties worden opgevolgd
  • Padding, margin en border properties hebben invloed op hoe ver de box van andere elementen af staat.

Inline boxes

  • Een inline box begint niet op een nieuwe regel
  • width en height properties worden niet toegepast
  • Padding, margins en borders worden toegepast op andere telementen. Alleen verticaal heeft zo'n box geen invloed op andere inline elementen.
