Static Dit is de standaardwaarde. Elementen verschijnen achter elkaar, zoals de flow gaat in het html document.
absolute Dit betekent dat een element een absolute positie heeft ten opzichte van het ouder-element. Dus het volgt niet de flow van het document, maar kan ook met andere elementen overlappen.
fixed Element is gepositioneerd relatief gezien aan het browser scherm.
Relative Element is gepositioneerd relatief aan zijn oorspronkelijke positie.
sticky Element is gestickied aan het browserscherm, en zal dus ook mee scrollen.
