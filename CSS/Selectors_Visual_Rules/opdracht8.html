Cascade

CSS staat voor Cascading Style Sheet. Vertaald betekent dat 'trapsgewijs'. Dit betekent dat, net als bij een trap, alles stap voor stap gedaan wordt. Als twee CSS regels op dezelfde elementen invloed hebben betekent dat de laatste style wordt toegepast. Dus als je twee keer een kleur doorgeeft aan een div element, wordt de laatst aangegeven kleur toegepast.

Inheritance

Als je regels opstelt, worden deze door alle child-elementen overgenomen. Alle header en paragraaf elementen in deze div nemen de kleur over die de div heeft doorgegeven. Deze dikgedrukte tag zit in een 'strong' tag, maar is nog steeds grijs. Deze dikgedrukte tekst is dan weer zwart, omdat ik heb aangegeven dat tekst in een 'span' element zwart en dikgedrukt moet zijn.
