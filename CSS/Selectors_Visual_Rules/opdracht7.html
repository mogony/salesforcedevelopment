  • General selectors
    • Grouping selector. Alle list items 'li' zijn nu bold
    • Universal selector. ALLE tekst op de pagina heeft nu lettergroote 18px
    • Op class, dmv een punt. Dit list item heeft klasse "klasse" die deze tekst schuingedrukt maakt
    • Op Id, dmv #. Dit list item heeft een id genaamd "ajdie" die deze tekst 'normaal' maakt.
  • Combinator selectors. De volgende unorganized list heeft klasse "cs"
    • descendant selector, hierdoor zijn alle list item elementen genest in de div class "cs" nu monospace
    • child selector, hierdoor zijn de list items die direct kinderen zijn in de div class "cs" roze.
      • Dit is niet roze
      • Maar dit wel, omdat het list item hierboven id "adjss" en dmv een adjacent sibling selector '+' de kleur roze maakt
      • Dit list item en hierboven zijn schuingedrukt dmv een general sibling selector (~) van het eerste list item
  • Pseudo-class selectors
    • Door een pseudo-class heeft deze link nu andere kleuren dan normaal als je er met je muis over heen beweegt
  • Pseudo-element selectors
    • Dit list item heeft klasse "pes". Met pseudo-element selector "first-letter" is de eerste letter rood gemaakt.
