• Type selectors: Selecteert specifieke html tags.
    • h1 { ... }
  • Descendant selectors: Begint ook met een specifieke html tag, maar daarna volgt nog een html tag. Hier wordt dus een html tag geselecteerd als die alleen voorkomt in de daarop voorgaande html tag.
    • article p { ... }
  • Class Selectors: Selecteert alle html tags die in een bepaalde klasse vallen.
    • .dit-is-een-klasse { ... }
