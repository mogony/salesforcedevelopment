Css begint met een selector: dit geeft aan waar de code die volgt toepasbaar op zal zijn. Vervolgens een krulhaak. De krulhaak geeft aan dat nu de styling begint. De styling begint met het type styling dat moet worden toegepast. Dat kan zijn van font type tot de kleur van een element. Hierna volgt een dubbele punt om aan te geven dat het volgende wat komt de waarde is van de styling. Font type moet dik zijn, of de kleur moet rood zijn. Dit wordt afgesloten met een puntkomma.
